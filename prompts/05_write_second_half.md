# 【展開の絶対ルール：逆張りと共感のターン】
後半の展開では、単なる理論の解説（Information）で終わらせないでください。解決策を提示する前に、必ず以下の展開を挟んでください。

「世間一般では〇〇と言われている。**しかし、私たちのような気にしすぎる人間（INFJ）にとって、それは残酷な事実だ（あるいは、だから苦しいのだ）。**」

このように、一度一般論を否定し、読者の「言葉にできなかった痛いところ」を鋭く指摘（言語化）してください。読者が「まさに私のことだ！」と痛感するような生々しさを意識してください。

# 結論の書き方
ただの精神論（頑張ろう、気にしないようにしよう）は禁止です。
INFJの気質を肯定した上で、「じゃあ明日からどう考えれば（どういう解釈のフィルターを持てば）心が少し軽くなるのか」という具体的な処方箋を渡して終わってください。

# 執筆モードを【Mode B: 静かなる着火】に切り替え
・トーン: 熱血教師のような「説教」や、インフルエンサーのような「成功法則の提示」は禁止。あくまで「同志としての提案」。
・「〜すべきだ」ではなく「〜してみるのはどうだろうか」「〜という視点もある」と、選択肢を置くように語りかける。
・読後感: 読者が「ふと、スマホを置いて少し考えてみようかな」と静かに行動を始めたくなる温度感。
