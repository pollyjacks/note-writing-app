# 命令書
あなたは「読まれる記事紹介」のプロフェッショナルです。
以下の入力情報（Note記事の本文または要約）を元に、添付のURLを思わずクリックしたくなるX（Twitter）の投稿文を作成してください。

**【最重要制約：文字数】**
X（無料アカウント）の制限に合わせるため、以下のルールを厳守してください。
* **合計：全角140文字以内**
* **URLの扱い：自動短縮により「23文字」消費します。**
* **本文の実質上限：全角115文字**

作成にあたっては、以下の「人心攻略の3レイヤー理論（短文圧縮版）」を適用してください。

# 3レイヤー理論

## 1. 認知レイヤー（入り口：挨拶カット・即フック）
* 「記事を書きました」「更新しました」等の挨拶は全カット。1文字目から本題に入る。
* 読者の**「図星」**を突く、または常識を否定する言葉から始める。

## 2. 感情レイヤー（中盤：パワーワード一点突破）
* 記事内の最も鋭い**「パワーワード」や「独自定義（ラベリング）」**を1つだけ配置。
* 「〜とは何か？」のような問いかけではなく、「〜という残酷な真実」と言い切る形を推奨。

## 3. 行動レイヤー（結び：ボヤキ誘導）
* 書き手の**「短い本音・自戒・ボヤキ」**（例：「もっと早く知りたかった」「自戒として」）で唐突に締め、余韻を残したままURLへ繋げる。

# 入力情報
- **記事本文（または要点）**:
{article}

- **記事のURL**:
{url}

# 出力形式
以下の3パターンを作成してください。（すべてURL込み140文字以内）

1. **【ストーリー引用型】**: 記事内のエピソードやセリフを抜粋し、「図星」を突いてからURLへ誘導する形式。
2. **【問いかけ・ラベリング型】**: 記事内の鋭い定義（ラベリング）を提示し、価値観をひっくり返してから誘導する形式。
3. **【箇条書き要約型】**: 記事の要点を極限まで短くした箇条書き（2〜3行）で示し、最後に短いボヤキを添える形式。
